Опис проєкту

Цей проєкт реалізує шаблонний метод (Template Method Pattern) для експорту даних користувачів, отриманих з API https://jsonplaceholder.typicode.com/users, у різні формати: CSV, JSON та XML.

Основна логіка експорту інкапсульована в базовому класі DataExporter, а специфічне форматування реалізується у підкласах.

Структура проєкту
src/exporters — класи експорту (CSV, JSON, XML)
src/data — типи даних користувача
src/iterators — ітератори для обходу експортованих файлів
src/main.ts — демонстрація експорту
src/main-iterate.ts — демонстрація ітераторів

Запуск проєкту
Встановлення залежностей:
npm install

Запуск експорту:
npx ts-node ./src/main.ts

Запуск ітераторів:
npx ts-node ./src/main-iterate.ts

Опис патерну
Шаблонний метод визначає загальний алгоритм експорту даних у базовому класі. Окремі кроки, такі як форматування та збереження, реалізуються у підкласах.
Це дозволяє уникнути дублювання коду та забезпечує розширюваність системи без зміни основної логіки.
